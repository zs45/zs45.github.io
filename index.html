Intro

My name is Zachary Schrock. I am an avid and passionate software engineer, looking to bring my ideas and thought processes on all things technology, software and architecture into the real world, one line of code at a time. The purpose of this site is to both describe and showcase my projects and abilities with various technologies. Side Projects

Red Light, Green Light

My senior design project, "Red Light, Green Light", was a year long project developed by myself and three of my classmates. Our client was a professor in our school's psychology department, and we developed a video game for her to use in research experiments, that was similar in design to the children's game with the same name.

The game was created in the Unity game engine, with scripts written in C# and Python. (Original executable and .dll files gone with old laptop, code is on private GitHub repo I maintain access too)


SideHero

A smaller scale project, SideHero was a video game that was a lot of fun creating. Developed on Xcode for Mac, the game is a 2D side scroller inspired by games such as Super Mario and Mega Man. The game was developed using C++ and OpenGL.
